# Tipply Stream Deck

Tipply Stream Deck is a simple plugin for controlling your Tipply donations from Stream Deck.

## What it does

- resend the latest donation
- skip the current donation alert
- toggle donation alerts
- toggle alert sound
- toggle moderator mode

## Install and use

1. Install the plugin in Stream Deck.
2. Add one of the Tipply actions to a key.
3. Open the action settings and paste your Tipply `auth_token` cookie.
4. Wait for the connection status to turn green, then use the button.

If your session expires, paste a fresh `auth_token` cookie from an active Tipply session.

## Requirements

- Stream Deck app
- a Tipply account
- macOS or Windows
